OGC Building Blocks Meta-Registry

A central index that maps short aliases (e.g. @ogc/main) to OGC Building Blocks register URLs, so that bblocks-config.yaml imports stay stable even when registers move hosts.

  • https://w3id.org/ogc/bblocks/meta-register.json — alias index (use this in tooling)
  • https://w3id.org/ogc/bblocks/meta-register-orgs.json — organisation metadata
  • index.json / orgs.json — this deployment's compiled artifacts
  • GitHub repository — source data and governance
